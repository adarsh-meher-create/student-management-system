# 🎓 Student Management System

A menu-driven Student Management System built with **Python**, **Object-Oriented Programming (OOP)**, and **SQLite**. This application allows users to manage student records efficiently through a simple command-line interface.

---

## 🚀 Features

- ➕ Add new students
- 📋 View all students
- 🔍 Search students by:
  - Name
  - Age
  - Course
- ✏️ Update student details:
  - Name
  - Age
  - Course
- 🗑️ Delete students with confirmation
- 📊 Count total students
- 🔄 Sort students by:
  - Name (A–Z)
  - Name (Z–A)
  - Age (Low → High)
  - Age (High → Low)
- ✅ Input validation
- 💾 Data stored permanently using SQLite
